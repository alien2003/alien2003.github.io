Senior DevOps / SRE, more than ten years on the job, working remotely.

Day-to-day that's AWS, Kubernetes (mostly EKS), Terraform, and
whichever observability stack happened to already be there.
Production wants to be up. Pipelines want to ship. Postgres wants
its WAL retention sized right. Somebody has to make sure none of
that slips.

This is where some of it gets written down. Long postmortems,
opinions about tools I keep arguing with, notes I want to be able
to find again at 03:00 the next time the same class of failure
shows up.
